Actúa como un Senior Product Designer + Senior Frontend Developer con 8+ años de experiencia diseñando landing pages para negocios de salud.

Diseña una landing page para una óptica especializada en:

- Salud visual
- Venta de monturas modernas
- Brigadas visuales empresariales
- Exámenes visuales para pase de conducción

El diseño debe transmitir confianza médica + modernidad tecnológica.

Estilo visual:
- Minimalista moderno
- Glassmorphism sutil y elegante (no exagerado)
- UI limpia y profesional
- Sensación light y luminosa
- Espacios amplios y jerarquía clara

Colores de acento:
- Turquesa brillante: #00F0FF
- Amarillo vibrante: #FFFF00

El fondo debe ser claro (blanco o gris muy suave).
El glass debe tener blur suave, sombras delicadas y bordes redondeados premium.

Estructura obligatoria:

1. Hero Section
   - Headline potente y profesional
   - Subtítulo enfocado en salud visual y confianza
   - CTA principal: “Agenda tu examen”
   - CTA secundario: “Conoce nuestras monturas”
   - Imagen profesional (no stock genérico cliché)

2. Servicios
   - Examen visual
   - Monturas modernas
   - Brigadas empresariales
   - Certificado visual para conducción

3. Sección Brigadas Visuales
   - Enfoque B2B
   - Beneficios para empresas
   - CTA para cotización

4. Catálogo destacado
   - Cards con efecto glass
   - Monturas premium

5. Testimonios
   - Diseño limpio y elegante

6. Footer profesional
   - Información de contacto
   - Redes sociales
   - Horarios

Requisitos técnicos:
- Layout basado en grid profesional
- Tipografía moderna tipo Inter o SF Pro
- Microinteracciones suaves
- Nada sobrecargado
- Que parezca diseñado por alguien senior, no por un junior ni un template básico.

Incluye también el copywriting profesional de cada sección.
